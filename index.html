매직 워드



문법설명
___404TEXT___문서를 찾을 수 없을 때 나오는 설명을 표시합니다.
___PAGERAWNAME___문서의 제목을 표시합니다.
___PAGENAME___문서의 제목을 필터링하여 표시합니다.
___PAGECREATED___문서가 만들어진 시각을 표시합니다.
___PARENT___문서의 상위 문서를 표시합니다.
___PAGEACL___문서의 ACL을 표시합니다.
___VERSION___사이트에 사용된 엔진의 버전을 표시합니다.
___SITENAME___사이트의 이름을 표시합니다.
___TIMEZONE___사이트의 시간대를 표시합니다.
___LANGUAGE___사이트의 언어를 표시합니다.
___BR___줄바꿈 문자를 표시합니다.
___NOW___타임 스탬프를 표시합니다.
___DATETIME___타임스탬프를 깔끔하게 정리하여 표시합니다.
___ADDRESS___방문자의 IP 주소를 표시합니다.
___YOU___방문자의 닉네임 또는 IP 주소를 표시합니다.
